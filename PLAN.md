# Checkapp Plan

## Fertig

- Einfache Checklisten-Webseite erstellt
- Aufgaben hinzufügen, abhaken und löschen
- Checklisten erstellen
- Fortschritt anzeigen
- Neue Aufgabe und neue Checkliste über Knöpfe in der oberen Leiste öffnen
- Erledigte Aufgaben aus der Hauptliste ausblenden und über Archiv anzeigen
- Beispiel-Aufgaben im Archiv mit Datum älter als eine Woche hinzugefügt
- Optionales Fälligkeitsdatum für Aufgaben hinzugefügt
- Lokales Hosting und Autostart-Webserver entfernt
- App-Dateien als echte Repository-Dateien wiederhergestellt
- Sichere Löschfunktion mit Bestätigung und Papierkorb hinzugefügt
- Papierkorb löscht Aufgaben nach 3 Tagen automatisch
- Aufgaben können aus dem Papierkorb wiederhergestellt werden
- Veröffentlichung auf GitHub vorbereitet
- Checklisten können umbenannt werden
- Checklisten können gelöscht werden
- Archiv kann gefiltert und sortiert werden
- Offene Aufgaben werden nach Fälligkeitsdatum sortiert
- Papierkorb optisch verfeinert
- Speicherung im Browser

## Als Nächstes

- GitHub Pages für das Repository Checkapp aktivieren
- Stabilität auf dem iPad weiter beobachten

## Ideen Später

- Prioritäten für Aufgaben
